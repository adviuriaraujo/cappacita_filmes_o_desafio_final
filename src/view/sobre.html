O que é o Cappacita Dev?

Cappacita Dev é o segundo curso gratuito de capacitação oferecido pela D1 Smarkio através de sua Comunidade Cappacita.

Enquanto o primeiro curso (Cappacita UXW) foi centrado em experiência do usuário e Design de Diálogos para interfaces conversacionais, nossos queridos chatbots, a nova edição é voltada para qualificar desenvolvedores.

A partir de uma introdução à lógica de programação, o curso nos levou em uma jornada de aprendizado e desafios. Nela, faprendemos sobre Javascript, JSON, NodeJs, Git, MySQL, HTML, CSS, além de experimentar diversas ferramentas para facilitar o desenvolvimento, como VS Code, Postman, MySQL Workbench, GitHub, entre outras.

Claro, não posso deixar de destacar nossa experiência com o ALTU, a plataforma exclusiva de desenvolvimento de chatbots da D1 Smarkio. Seus diferenciais estão nas formas como a plataforma melhora a experiência dos usuários, apresentando uma visualização intuitiva dos fluxos conversacionais no próprio ambiente de desenvolvimento, além de possuir uma documentação detalhada e acessível.

Durante o Módulo 2 do curso, desenvolvi um chatbot educativo que fala sobre a LGPD, privacidade e proteção de dados pessoais. Quer dar uma conferida?

Qual foi o Desafio Final?

Para a conclusão do curso, recebemos o desafio de desenvolver uma interface gráfica com filmes ou séries através da API do TMDB. Para tanto, deveríamos criar uma API própria para consumir as APIs do TMDB. Outros requisitos foram preparar o Backend com NodeJS e incluir uma ligação com um banco de dados MySQL. No final, nosso projeto ficará disponibilizado no GitHub.
